How it works

01

Competitor scraping

We query the public Meta Ad Library API for competitor brands in your niche. All data retrieved is publicly available ad transparency data — no private user data is accessed.

02

Scoring & analysis

Each ad is scored based on run duration, format, creative patterns, and engagement signals. Our pipeline identifies the top-performing creatives from high-spending brands.

03

Brief generation

Winning ads are transformed into creative briefs, scripts, and shot lists — ready for your creative team to act on immediately.

04

Momentum monitoring

Continuous scanning detects new high-spend creatives as they emerge, so your team always has the freshest competitive intelligence.

Privacy Policy

Last updated: April 2026

What data we collect

Creative Intelligence Engine collects only the minimum data necessary to provide the service:

  • Account information (name, email address) when you create an account
  • Usage data (analysis runs, feature usage) to improve the product
  • Public ad data retrieved from the Meta Ad Library API on your behalf

We do not collect, store, or process any personal data from Meta users or ad audiences. All Meta Ad Library data we retrieve is publicly available ad transparency data.

How we use Meta API data

We use the Meta Marketing API exclusively to retrieve publicly available ad creatives and metadata from the Meta Ad Library. Specifically:

  • We query ad creatives, copy, and metadata for specified competitor brands
  • We use read-only access — we never create, modify, or delete ads
  • Retrieved data is used solely for competitive intelligence analysis within our platform
  • Data is cached for up to 72 hours to minimize API calls, then deleted
  • We do not share Meta API data with third parties
  • We do not use Meta API data for ad targeting or audience building

Data storage & security

Your data is stored securely on servers within the EU. We use industry-standard encryption in transit (TLS) and at rest. Access to production data is restricted to authorised personnel only.

Analysis results and cached ad data are automatically purged after 72 hours. Account data is retained for the duration of your subscription and deleted within 30 days of account closure.

Third-party services

We use the following third-party services to operate the platform:

  • Meta Graph API — to retrieve public Ad Library data
  • Anthropic Claude API — to generate creative briefs and analysis
  • ElevenLabs — to generate voiceover scripts
  • OpenAI Whisper — for audio transcription of ad creatives

Each service has its own privacy policy. We share only the minimum data necessary for each service to function.

Your rights

You have the right to access, correct, export, or delete your personal data at any time. To exercise these rights, contact us at the address below. We will respond within 30 days.
